# QUIZ & RUN !!!!!!

## ゲームの内容
障害物を避けながら、クイズにも正解し、スコアを重ねてクリアをめざせ！

ただのランゲームではありません。途中にクイズが出現します！

頭を使いながら、障害物を避け、100秒以内に130スコアをめざしましょう！

## 操作説明
- 左右矢印キー、若しくはAとDキーで左右に移動するのみ！
